## 전역객체(Global object)
- window
- 특수한 객체
- 모든 객체(변수, 함수)는 전역객체(window)의 프로퍼티, 메소드다.
```
function func(){
  alert('Hello?');    
}

func();
window.func();
```
> 둘다 실행 됨<br/>모든 전역변수와 함수는 window 객체의 프로퍼티지만 암시적으로 window를 생략한것이다.<br/>즉, func();는 window의 메소드인 것

- 웹브라우저(익스프로우, 크롬...)는 전역객체로 window이고, node.js 에서는 global이 전역객체이다.
